# NLP - Natural Language Processing

---

## 1. 上层应用

+ 文本分类（如：情感/倾向分析、恶意xx检测）
+ 回归任务（如：股价预测、金融评估）
+ 信息抽取（如：命名实体识别、关系识别/抽取、实体消歧、指代消解）
+ 信息摘要（如：自动文摘）
+ 对话系统（如：语音助手、智能客服）
+ 机器翻译（双语、多语翻译）
+ 阅读理解（开放域 or 限定域）
+ 自然语言推断（含：语义相似、文法解析、文本蕴含、关联分析...）
+ 知识挖掘、知识图谱
+ 推荐系统、广告投放
+ 观点/意图挖掘分析
+ 事件抽取与跟踪预测
+ 智能创作（生成各种产物）
+ 风格转换（文字/图像/语音均有风格转换任务）
+ 多模态转换（文字/图像/语音之间互相转换）
+ 可解释机器学习
+ 自动机器学习 autoML
+ ...

## 2. 中层方法

+ 语言模型 Language Model、预训练语言模型 Pre-Training Language Model
+ 词句嵌入 Embedding、表示学习 Learning Representation、解耦表示 Disentangle Representation
+ 元学习 Meta Learning、小样本学习 Few-Shot Learning、零样本学习 Zero-Shot Learning
+ 增量学习 Augmentation Learning、终生学习 Life-Long Learning、在线学习 Online Learning
+ 无监督学习 Unsupervised Learning、半监督学习 Semi-Supervised Learning
+ 自监督学习 Self-Supervised Learning、远程监督 Distant Supervision
+ 迁移学习 Transfer Learning、多任务学习 Multi-Task Learning
+ 自编码器 Auto-Encoder、变分自编码器 Variational Auto-Encoder
+ 图神经网络 Graphic Neural Network
+ 神经图灵机 Neural Turing Machine
+ 模型压缩 Network Compression
	+ 网络剪枝 Network Pruning
	+ 知识蒸馏 Knowledge Distillation
	+ 参数量化 Parameter Quantization
	+ 少参数模型设计 Architecture Design
	+ 低秩逼近 Low Rank Approximation
	+ 动态计算 Dynamic Computation
+ ...

## 5. 经典机器学习

+ 信息论 Information Theory、统计学 Statistics
+ 贝叶斯理论 Bayesian Theory、贝叶斯网络 Bayesian Network
+ 支持向量机 SVM、核方法 Kernel Method
+ 高斯混合模型 Gaussian Mixture Model、高斯过程 Gaussian Process、变分估计 Variational Approximation
+ 时间序列分析、隐马尔可夫模型 Hidden Markov Model、条件随机场 Conditional Random Field
+ 采样方法
	+ 基本采样法 Basic Sampling
	+ 拒绝采样 Rejection Sampling
	+ 重要性采样 Importance Sampling
	+ MCMC 采样 Markov Chain Monte Carlo
	+ M-H 采样方法 Metropolis-Hastings
	+ 吉布斯采样 Gibbs Sampling
	+ Slice Sampling
+ 经典算法：
	+ Decision Tree（ID3、C4.5、CART、...）、Random Forest、GDBT
	+ K-Means
	+ SVM（Support Vector Machine）
	+ Apropri (Association Rule Mining)
	+ EM（Expectation Maximization）
	+ PageRank
	+ AdaBoost、以及各种集成方法 Ensemble
	+ kNN（k-Nearest Neighbors）、以及各种聚类方法
	+ Naive Bayes
	+ LR（Logistic Regression）
	+ SVD、PCA、LSI、PLSI
	+ LSA、PLSA、LDA
	+ Graphical Model（Bayesian Network、Factor Graph、CRF...）
+ ...
